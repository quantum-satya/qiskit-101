# Contributing Guidelines

Thank you for your interest in contributing to **Qiskit 101**!

## Getting Started
1. Fork this repository and clone your fork locally.
2. Create a new branch for your changes:

git checkout -b feature-name

3. Activate your virtual environment:

source ../qiskit-env/bin/activate

4. Make your improvements â€” fix bugs, add notebooks, or enhance documentation.

## Submitting Changes
1. Run and test your notebooks before committing.
2. Commit with a clear message:

git commit -m "Add Grover algorithm notebook"

3. Push your branch and open a **Pull Request** on GitHub.

## Notes
- Follow the Qiskit coding style.
- Explain your work clearly in the notebook.
